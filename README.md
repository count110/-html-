
🧪 XRD 数据对比与可视化工具

这是一个轻量级的 XRD（X-ray Diffraction）数据对比与可视化应用，面向材料、化学与环境相关科研人员，支持多组 XRD 曲线的交互式叠加展示与快速对比分析。

本工具基于 Plotly.js 构建，无需复杂环境配置，打开即用，适用于相鉴定、样品结构对比、合成条件差异分析等典型科研场景。

✨ 主要特性

📊 多组 XRD 数据交互式叠加对比

🔍 支持缩放、平移与峰位观察

⚡ 前端运行，响应快速、操作直观

🧩 适合科研汇报、论文作图与教学演示

如果你正在寻找一个简单、直观、可定制的 XRD 对比工具，这个项目可以作为你的即用型解决方案或二次开发基础。
